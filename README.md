# CypressTestingSuite

Фреймворк автоматизации тестирования корпоративных веб-приложений с использованием актуальной версии Cypress.

---

## Обзор

CypressTestingSuite ориентирован на построение стабильного и масштабируемого решения для end-to-end тестирования веб-приложений.

Проект разрабатывается с учетом поддерживаемости, контроля зависимостей и возможности дальнейшего расширения, включая потенциальную поддержку TypeScript.

---

## Возможности

* UI / E2E тестирование
* API тестирование с использованием `cy.request()`
* Реализация подхода Page Object Model (POM)
* Параметризованные тесты на основе внешних `.json` данных
* Расширенные примеры и дополнительные практические задачи
* Структурированный подход к разработке автотестов
* Поддержка обновления зависимостей
* Готовность к интеграции в CI/CD процессы

---
## ⚙️ Требования

- Node.js **22** или другая версия с поддержкой **ES-модулей**

## Установка

```bash
npm install
```

---

## Запуск тестов

### Запуск в интерактивном режиме

```bash
npx cypress open
```

### Запуск c отображением браузера

```bash
npx cypress run --headed
```

---

## Управление зависимостями

Проект использует актуальную стабильную версию Cypress и предполагает контролируемое обновление зависимостей с проверкой совместимости.

---

## Цель

Создание поддерживаемого и масштабируемого фреймворка автоматизации тестирования для корпоративных веб-платформ с ориентацией на production-уровень.
