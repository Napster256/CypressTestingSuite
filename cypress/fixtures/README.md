# fixtures/

Директория для хранения тестовых данных.

Фикстура — это код, который подготавливает окружение/данные/ресурсы для теста (setup), и при необходимости — очищает/освобождает их после теста (teardown). Фикстуры инъектируются в тесты, чтобы повторно использовать подготовку без дублирования.

Используется для:
- Параметризации тестов
- Data-driven подхода
- Хранения JSON-структур API
- Mock данных

Файлы доступны через:
`cy.fixture('fileName.json')`
