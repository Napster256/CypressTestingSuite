# the-first-steps

Вспомогательные модули, относящиеся к базовым учебным сценариям.

Назначение:

- Поддержка тестов из раздела e2e/the-first-steps
- Демонстрация принципов декомпозиции логики
- Формирование правильной структуры проекта с первых шагов

Содержит:

- Простые утилиты
- Вспомогательные функции для базовых тестов
- Примеры вынесения логики из spec-файлов

Цель раздела:

Показать переход от примитивных тестов к более структурированному подходу через разделение ответственности.
