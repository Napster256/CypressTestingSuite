# cypress

Основная директория тестового фреймворка Cypress.

Содержит:

- e2e — тестовые спецификации
- fixtures — тестовые данные
- support — вспомогательные команды и конфигурации
- downloads — загруженные файлы (создаются автоматически)
- screenshots — скриншоты при падениях тестов
- videos — записи прогонов тестов

Структура соответствует стандарту Cypress
